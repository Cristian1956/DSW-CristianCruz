Etiquetas básicas de HTML

Las Etiquetas basicas de HTML son fragmentos de texto que se usan para dar propiedades especificas al texto cuando escribimos en codigo HTML.

Hay una serie de etiquetas que son las más usadas para crear cualquier documento HTML, las cuales son:

  • body : Detallo el cuerpo del documento
  • head : Determina el encabezado del documento
  • div : división dentro del contenido
  • a : para enlaces
  • strong : para poner el texto en negrita
  • br : para saltos de línea
  • H1…H6 : para títulos dentro del contenido
  • img : para añadir imágenes al documento
  • ol : para listas ordenadas, ul: para listas desordenadas, li: para elementos dentro de la lista
  • p : para parágrafos
  • span : para estilos de una parte del texto

Atributos

Estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios

Atributos comunes :

  • id : proporciona un identificador único en el ámbito de todo el documento para un elemento
  • class : proporciona una manera de clasificar elementos similares
  • style: para un elemento particular
  • title : se suele usar para unir explicación subtextual a un elemento
